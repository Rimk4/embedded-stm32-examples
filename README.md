## Как использовать:

1. **Сборка проекта:**
```bash
make DEBUG=1          # Сборка с отладочной информацией
make                  # Релизная сборка
```

2. **Прошивка:**
```bash
make flash            # Прошить через ST-Link
```

3. **Отладка:**
```bash
make debug            # Запустить GDB
```

4. **Анализ:**
```bash
make symbols          # Посмотреть ключевые символы
make disasm           # Сгенерировать листинг ассемблера
```

## Ключевые особенности:

1. **Двухэтапная загрузка:** `Starter_Handler` → копирование → `Reset_Handler` → `main`
2. **Разделение LMA/VMA:** Код линкуется во Flash, но выполняется из SRAM
3. **Автоматическое копирование:** `.data` секция автоматически копируется из Flash в RAM
4. **Полная отладка:** Поддержка GDB с точками останова
5. **Гибкая конфигурация:** Можно включать/выключать отладочную информацию
